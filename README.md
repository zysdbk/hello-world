# 简介
基于layui+springboot+mybatis+redis的各种常用功能开发


# 主要功能
单点登录</br>
用户验证</br>
权限控制</br>
用户注册</br>
表单验证</br>
表格排序，搜索，crud,excel导出</br>
文件图片上传下载，多文件上传下载</br>
